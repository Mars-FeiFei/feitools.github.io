Parser 概述

FeiSharp Parser 是一个用于解析和执行 FeiSharp 脚本语言的解析器。它支持变量声明、函数定义、控制流、类系统以及多种内置功能。

注意： FeiSharp 是一种动态类型脚本语言，主要用于自动化任务和快速原型开发。 
public class Parser
{
    // 解析器的主要入口方法
    public void ParseStatements(string funcName = "")
    {
        // 解析各种语句
    }
}

架构设计

Parser 采用递归下降解析器设计模式，包含以下核心组件：

核心数据结构

  • _tokens: 词法分析器生成的令牌列表
  • _variables: 存储变量名和值的字典
  • _functions: 存储函数信息的字典
  • _classInfos: 存储类信息的字典
  • _results: 存储函数返回值的字典

内置常量

Parser 初始化时自动创建以下常量：

_variables.NewAdd("true", true);
_variables.NewAdd("false", false);
_variables.NewAdd("null", null);
_variables.NewAdd("pi", Math.PI);
_variables.NewAdd("e", Math.E);
_variables.NewAdd("tau", Math.Tau);
_variables.NewAdd("buildVersion", "8.0");

关键字语法

FeiSharp 支持以下关键字，每个关键字都有特定的语法格式：

关键字 语法 描述
var var variableName = value; 声明变量并赋值
print print(text); 输出文本到控制台
init init(variableName, Type); 初始化变量为默认值
set set variableName = value; 修改变量值
func function funcName(parameters)fbegin: ... fend; 定义函数
if if(condition) ibegin ... iend 条件语句
while while(condition) wbegin ... wend 循环语句
示例：基础语法
// 变量声明和赋值;
var name = "FeiSharp";
var count = 10;

// 条件语句;
if(count > 5)
ibegin:
    print("Count is greater than 5");
iend;

// 循环语句;
while(count > 0)
wbegin:
    print(count);
    set("count", count - 1);
wend;

// 函数定义;
function add(a, b)
fbegin:
    _return a + b;
fend;

// 函数调用;
var result = add(5, 3);
print(result);

数学函数

FeiSharp 提供了一系列内置数学函数：

函数 语法 描述
sin sin(angle); 计算正弦值
cos cos(angle); 计算余弦值
tan tan(angle); 计算正切值
pow pow(base, exponent); 计算幂运算
sqrt sqrt(value, root); 计算根运算
rand rand(min, max); 生成随机数
示例：数学运算
// 计算幂运算;
var result = pow(2, 3);  // result = 8;

// 计算正弦值;
var sinValue = sin(0.5);  // sinValue = sin(0.5);

// 生成随机数;
var randomNum = rand(1, 100);  // 生成1-100的随机数;

// 计算平方根;
var root = sqrt(16, 2);  // root = 4;

字符串函数

字符串处理函数：

函数 语法 描述
strlen strlen(string); 获取字符串长度
strreplace strreplace(target, replacement); 替换字符串
substr substr(source, startIndex); 获取子字符串
strindexof strindexof(source, index); 获取指定索引的字符
cstr cstr(value, variableName); 转换为字符串
示例：字符串操作
// 字符串长度;
var length = strlen("Hello World");  // length = 11;

// 字符串替换;
var text = strreplace("Hello, world!", "Hello", "Hi");  // text = "Hi World";

// 子字符串;
var sub = substr("Hello World", 6);  // sub = "World";

// 类型转换;
cstr(123, "strValue");  // strValue = "123";

I/O 操作

文件读写和用户输入操作需要导入相应命名空间：

安全提示： 所有文件操作都会请求用户确认，确保安全性。
操作 语法 命名空间
read read(variableName as filePath); FeiSharp.IO
export export(content as filePath); FeiSharp.IO
readline readline(variableName); 内置
readkey readkey(variableName); 内置
示例：文件操作
// 导入命名空间
import "FeiSharp.IO";

// 读取文件内容
read("fileContent" as "data.txt");

// 写入文件
export("Hello World" as "output.txt");

// 读取用户输入
readline("userInput");

命名空间

FeiSharp 使用命名空间来组织功能模块：

命名空间 功能 启用关键字
FeiSharp.IO 文件输入输出 read, export
FeiSharp.Text.Json JSON 处理 getVarsFromJsonFilePath...
FeiSharp.Net 网络操作 gethtml...
FeiSharp.System 系统交互 sys...
$prelude.fsc FeiSharp标准内置库文件 sin...
$math.fsc FeiSharp标准数学库文件 sin...
$text.fsc FeiSharp标准文本处理库文件 strreplace...
示例：使用命名空间
// 导入命名空间;
import "FeiSharp.IO";
import "FeiSharp.Net";

// 现在可以使用相关功能;
read("data" as "file.txt");

// 获取网页内容;
gethtml("https://example.com", "htmlContent");
