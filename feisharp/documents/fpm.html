fpm 介绍

fpm (FeiSharp Package Manager) 是 FeiSharp 编程语言的官方包管理工具，用于安装、搜索、管理第三方库和应用程序。fpm 支持命令行操作（fpm.exe）和图形化界面（fpmui.exe），满足不同开发场景的需求。

📦 核心特性
  • 一键安装/卸载包
  • 多源仓库支持
  • 包信息查询
  • 依赖自动解析
🎯 适用场景
  • FeiSharp 项目依赖管理
  • SDK 与工具链扩展
  • 团队私有包共享
# 基本工作流
fpm install feisharp_vscode_extension      # 安装包
fpm search all console            # 搜索包
fpm list all                   # 查看已安装包
fpm remove feisharp_vscode_extension       # 卸载包

安装与配置

fpm 随 FeiSharp SDK 一起发布，安装 SDK 后自动可用。确保 %ProgramFiles%\FeiSharpSDK10 已添加到系统 PATH 环境变量中。

fpm source default    # 切换成默认源

📦 fpm install

安装一个或多个包到当前环境。

语法说明
fpm install <包名1> <包名2> ...安装指定包，支持同时安装多个。
示例: fpm install calcfunc feisharp_vscode_extension
实例
PS> fpm install calcfunc feisharp_vscode_extension
⬇️ 下载 calcfunc  ... 完成
⬇️ 下载 feisharp_vscode_extension v3.2.1 ... 完成
✅ 成功安装 2 个包

🔍 fpm search

按类型和关键字搜索包。

语法说明
fpm search <类型> <关键字>在指定类型中搜索。
示例: fpm search Math c
fpm search All <关键字>在所有类型中搜索。
示例: fpm search All feisharp
实例
PS> fpm search All console
CS_Console

ℹ️ fpm get

获取包的详细信息。

语法说明
fpm get <包名>.info显示包的完整信息。
示例: fpm get calcfunc.info
fpm get <包名>.version仅显示版本号。
示例: fpm get calcfunc.version
 
PS> fpm get calcfunc.info
名称: calcfunc
版本: 2.0.1
类型: fsc
依赖: $prelude
描述: 高性能计算库

🗑️ fpm remove

卸载已安装的包。

语法说明
fpm remove <包名1> <包名2> ...卸载一个或多个包。
示例: fpm remove feisharp_vscode_extension
⚠️ 卸载依赖其他包的组件时，fpm 会发出确认提示。

🌐 fpm source

配置包仓库源地址。

语法说明
fpm source <URL名>切换到指定名称的源（default为官方默认源）。
示例: fpm source default

📋 fpm list

列出已安装的包。

语法说明
fpm list all显示所有已安装的包（名称+版本）。
示例: fpm list all
输出示例
已安装包 (共 4 个):
├── feisharp_vscode_extension v3.2.1
└── calcfunc v0.9.2

🖥️ fpmui - 图形化包管理器

fpmui 是 fpm 的图形界面版本，提供直观的鼠标操作体验，适合不熟悉命令行的开发者或需要批量管理包的用户。

FeiSharp Package Manager UI
feisharp_vscode_extension v3.2.1
高性能VSCode插件
已安装
fpmui 基于 Windows Forms 开发，支持 Native AOT 编译，启动迅速，占用内存低。

✨ fpmui 功能模块

📦 包管理
  • 可视化浏览/安装/卸载包
  • 批量操作支持
  • 依赖列表形展示
🔎 智能搜索
  • 实时搜索过滤
  • 按类型/作者/标签分类
  • 热门推荐
⚙️ 仓库管理
  • 图形化添加/删除源
  • 源优先级调整
  • 镜像测速
📊 环境总览
  • SDK版本信息
  • 磁盘占用分析
  • 更新通知

🚀 fpmui 使用指南

  1. 启动：双击 fpmui.exe，主界面显示包列表和分类。
  2. 搜索/筛选：在顶部搜索框输入关键字，右侧可按类型（lib/tool/sdk）快速筛选。
  3. 安装包：找到目标包，点击“安装”按钮，确认依赖后自动下载安装。
  4. 卸载包：在“已安装”标签页，选中包后点击“卸载”。
  5. 切换源：工具栏“源设置”可管理仓库地址，支持临时切换以加速下载。
  6. 查看详情：双击任意包可查看完整信息（许可证、主页、依赖）。
权限提示： 安装包到系统目录（如 Program Files）时，fpmui 会自动请求管理员权限。

📦 打包规范（为开发者）

要发布自己的包到 fpm 仓库，需遵循 .fpm 包格式：

my-package/
├── package.json      # 包清单（名称、版本、依赖）
├── lib/               # 库文件（.dll / .fs）
├── tools/             # 可执行工具
└── docs/              # 文档
package.json 示例
{
  "name": "feisharp-example",
  "version": "1.0.0",
  "type": "Math",
  "dependencies": { "$io_extend.fsc": ">=1.0" },
  "author": "FeiSharp Team"
}

fpm source local   # 切换到官方默认源
