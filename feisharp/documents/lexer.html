Lexer 概述

FeiSharp Lexer 是 FeiSharp 编程语言的词法分析器，负责将源代码字符串分解为有意义的词法单元（Tokens）。它是编译器和解释器前端的重要组成部分。

注意： Lexer 是 FeiSharp 编译器/解释器的第一个阶段，将源代码转换为 Token 流供 Parser 使用。 
namespace FeiSharpStudio;

public class Lexer
{
    private readonly string _source;
    private int _index;

    public Lexer(string source)
    {
        _source = source;
        _index = 0;
    }

    public Token NextToken()
    {
        // 词法分析核心逻辑
    }
}

架构设计

Lexer 采用顺序扫描设计模式，逐个字符分析源代码并生成对应的 Token。

核心数据结构

  • _source: 待解析的源代码字符串
  • _index: 当前解析位置的索引
  • Token: 词法单元的数据结构，包含类型和值

处理流程

  1. 跳过空白字符（空格、制表符、换行符）
  2. 识别单个字符的标点符号和运算符
  3. 识别字符串字面量（双引号包围的内容）
  4. 识别数字字面量（整数和小数）
  5. 识别标识符和关键字
  6. 处理未知字符（输出警告信息）
  7. 返回 EndOfFile Token 表示解析完成

Token 类型

FeiSharp Lexer 识别以下类型的词法单元：

Token 类型 常量名 描述 示例
Punctuation TokenTypes.Punctuation 标点符号 ; , ( ) { } [ ] . : #
Operator TokenTypes.Operator 运算符 + - * / = < > ! & | ^ $
String TokenTypes.String 字符串字面量 "hello"
Number TokenTypes.Number 数字字面量 123, 3.14
Identifier TokenTypes.Identifier 标识符（变量名、函数名等） myVariable, calculateSum
Keyword （根据具体关键字） 语言关键字 var, function, if, while
EndOfFile TokenTypes.EndOfFile 文件结束标记 ""（空字符串）
Token 结构示例
// Token 类的基本结构
public class Token
{
    public TokenTypes Type { get; }
    public string Value { get; }
    
    public Token(TokenTypes type, string value)
    {
        Type = type;
        Value = value;
    }
}

字面量

字符串字面量

字符串字面量由双引号包围，支持任意字符：

// 字符串解析逻辑
if (current == '"')
{
    int start = ++_index;
    while (_index < _source.Length && _source[_index] != '"')
    {
        _index++;
    }
    return new Token(TokenTypes.String, _source[start.._index++]);
}
字符串示例
"Hello, World!"        // → Token(TokenTypes.String, "Hello, World!")
"FeiSharp v8.0"      // → Token(TokenTypes.String, "FeiSharp v8.0")
"123"                // → Token(TokenTypes.String, "123")

数字字面量

数字字面量支持整数和小数格式：

// 数字解析逻辑
if (char.IsDigit(current))
{
    int start = _index;
    while (_index < _source.Length && 
           (char.IsDigit(_source[_index]) || _source[_index] == '.')) 
        _index++;
    return new Token(TokenTypes.Number, _source[start.._index]);
}
数字示例
123           // → Token(TokenTypes.Number, "123")
3.14         // → Token(TokenTypes.Number, "3.14")
0.5          // → Token(TokenTypes.Number, "0.5")
1000         // → Token(TokenTypes.Number, "1000")

标识符和关键字

标识符规则

标识符必须以字母开头，可以包含字母和数字：

// 标识符解析逻辑
if (char.IsLetter(current))
{
    int start = _index;
    while (_index < _source.Length && char.IsLetterOrDigit(_source[_index]))
        _index++;
    string identifier = _source[start.._index];
    if (FeiSharpKeywords.KeywordMap.TryGetValue(identifier, out Token keywordToken))
        return keywordToken;
    return new Token(TokenTypes.Identifier, identifier);
}

解析流程

主循环逻辑

public Token NextToken()
{
    while (_index < _source.Length)
    {
        char current = _source[_index];
        
        // 1. 跳过空白字符
        if (char.IsWhiteSpace(current))
        {
            _index++;
            continue;
        }
        
        // 2. 检查单字符 Token（标点符号和运算符）
        if (current == '#') { _index++; return new Token(TokenTypes.Punctuation, "#"); }
        if (current == ':') { _index++; return new Token(TokenTypes.Punctuation, ":"); }
        // ... 其他单字符检查
        
        // 3. 检查字符串字面量
        if (current == '"') { /* 字符串解析逻辑 */ }
        
        // 4. 检查数字字面量
        if (char.IsDigit(current)) { /* 数字解析逻辑 */ }
        
        // 5. 检查标识符和关键字
        if (char.IsLetter(current)) { /* 标识符解析逻辑 */ }
        
        // 6. 未知字符处理
        Console.WriteLine("Unexpected character: " + current + ", it will be skipped");
        _index++;
    }
    
    // 7. 返回文件结束标记
    return new Token(TokenTypes.EndOfFile, "");
}

处理顺序

  1. 空白字符跳过：空格、制表符、换行符等
  2. 单字符Token识别：标点符号和运算符
  3. 字符串字面量：双引号包围的内容
  4. 数字字面量：整数和小数
  5. 标识符和关键字：字母开头的标识符
  6. 错误处理：未知字符警告

关键字识别

关键字通过 FeiSharpKeywords.KeywordMap 字典进行映射。这是一个静态字典，在程序启动时初始化。

// FeiSharpKeywords 类示例
public static class FeiSharpKeywords
{
    public static Dictionary KeywordMap = new()
    {
        { "var", new Token(TokenTypes.Keyword, "var") },
        { "function", new Token(TokenTypes.Keyword, "function") },
        { "if", new Token(TokenTypes.Keyword, "if") },
        { "while", new Token(TokenTypes.Keyword, "while") },
        { "return", new Token(TokenTypes.Keyword, "return") },
        { "print", new Token(TokenTypes.Keyword, "print") },
        { "true", new Token(TokenTypes.Keyword, "true") },
        { "false", new Token(TokenTypes.Keyword, "false") },
        { "null", new Token(TokenTypes.Keyword, "null") }
        // ... 其他关键字
    };
}
设计模式： 使用字典进行关键字映射比一系列 if-else 语句更高效，便于维护和扩展。

错误处理

Lexer 采用宽容的错误处理策略：

警告信息： 当遇到无法识别的字符时，Lexer 会输出警告信息并跳过该字符。 
// 未知字符处理逻辑
Console.WriteLine("Unexpected character: " + current + ", it will be skipped");
_index++;

错误处理策略

  • 宽容性解析：遇到未知字符时不抛出异常，而是跳过继续解析
  • 信息性警告：在控制台输出有用的错误信息
  • 位置保持：正确维护 _index 位置，确保后续解析正常
  • 可恢复性：错误不会导致解析过程中断

基础用法

创建 Lexer 实例

using FeiSharpStudio;

// 创建 Lexer 实例
string sourceCode = "var x = 10 + 20;";
Lexer lexer = new Lexer(sourceCode);

// 逐个获取 Token
Token token;
do
{
    token = lexer.NextToken();
    Console.WriteLine($"Type: {token.Type}, Value: {token.Value}");
} while (token.Type != TokenTypes.EndOfFile);
