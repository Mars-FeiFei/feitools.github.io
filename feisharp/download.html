安装指南

  1. 下载安装文件

    根据您的需求选择上述下载选项。首次使用建议下载完整安装程序。

  2. 运行安装程序

    Windows 用户运行 .exe 文件，macOS/Linux 用户解压 ZIP 包。

    # Windows FeiSharp_Installer.exe # macOS/Linux unzip feisharp.zip

  3. 配置环境变量

    将 FeiSharp 安装目录添加到系统 PATH 环境变量。

    # Windows (PowerShell) [Environment]::SetEnvironmentVariable("PATH", "$env:PATH;C:\Program Files\FeiSharp", "User") # Linux/macOS (Bash) echo 'export PATH="$PATH:/opt/feisharp/bin"' >> ~/.bashrc source ~/.bashrc

  4. 安装 VS Code 插件

    在 VS Code 中安装 FeiSharp 语言支持插件。

    # 通过下载的 .vsix 文件安装 code --install-extension feisharp-language-support-1.0.0.vsix

  5. 验证安装

    打开终端，运行以下命令验证安装是否成功。

    # 运行简单脚本 feisharp hello.fsc
