FeiSharp 语言设计简洁而强大，提供了现代脚本语言所需的所有功能

快速执行

轻量级解析器和即时执行引擎，启动迅速，适合快速脚本任务和自动化工作流。

简洁语法

直观的关键字和表达方式，降低学习成本，提高开发效率。无需复杂的类型声明。

模块化设计

支持命名空间和模块导入，功能按需加载，保持核心轻量，扩展性强。

安全优先

文件操作和系统交互均需用户确认，防止未经授权的访问，保护用户数据安全。

丰富标准库

内置数学函数、字符串处理、I/O操作、网络请求等常用功能，开箱即用。

跨平台

// FeiSharp 示例脚本;
import "$math.fsc";
// 变量声明和基本运算;
var name = "FeiSharp";
var version = 8.0;

// 输出到控制台;
print("欢迎使用 " + name + " 版本 " + version);

// 条件判断;
if(version > 7.0) 
ibegin:
    print("这是最新版本！");
iend;

// 循环示例;
var count = 1;
while(count <= 5)
wbegin:
    print("计数: " + count);
    set("count", count + 1);
wend;

// 函数定义和调用;
function greet(person)
fbegin:
    return "你好, " + person + "!";
fend;

var message = greet("开发者");
print(message);
