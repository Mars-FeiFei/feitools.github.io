FeiSharp 编程语言

现代、高效、易学的编程语言

FeiSharp 是一门专为提高开发效率设计的现代编程语言，结合了脚本语言的灵活性和系统语言的性能。

语言特性：

  • 简洁的语法，学习曲线平缓
  • 强大的类型推断系统
  • 内置并发支持
  • 与C#/Java无缝互操作
  • 跨平台运行时

示例代码：

// FeiSharp 示例
using namespace System.Console.Output, System.Math.Operators;
set main.point = main();
function:int main() {
    let nums = DATA[int]:[1, 2, 3, 4, 5];
    let squares = nums.Map(x => x ^ 2);
    Console.Write($"平方数: {squares}");
    return 0;
}
